Joystick

How to remote control almost anything.

This is a modular system that allows a user to remotely control things with a joystick.
Plug anything in and you can control it with the joystick: lights, fans, things that turn on and off, things that move...

We're used to this type of joysticks as a mean to play arcade games and have fun experiences. Here, I'm trying to take advantage of those expectation and reapply them in a new and unexpected context, by using the joystick to activate everyday object and create a playfulness out of that interaction.
